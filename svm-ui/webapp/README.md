## 环境初始化
```
npm config set registry http://registry.npm.taobao.org/
npm install vue2-dragula
npm install
```

## 配置文件修改
* 复制.env.development.example文件，去掉.example后缀
* 复制.env.test.example文件，去掉.example后缀
* 注意是复制文件，原来的.example扩展名文件还保留

## 运行开发环境
```
npm run dev
```

## 运行测试环境
```
npm run test
```

## 工程打包
```
npm run build
```
